Documentation
=============

The Hubbard-I solver approximates the solution of the impurity model by neglecting any hybiridization and solving an atomic problem. The atomic problem is defined by the local interaction Hamiltonian and the orbital dependent high-frequency behavior of the non-interacting bath Green's function.
